# DESIGN.md

## 气质与意象
安全监控中枢——如同机场安检X光屏幕：深色背景降低视觉疲劳，荧光绿/琥珀色/红色信号灯在暗场中即时传达风险等级。冷静、专业、零容忍。

## 视觉策略
- 图形语言：等宽数据网格、状态徽章、环形进度、脉冲动画表示实时扫描
- 图像方向：无装饰性插图，纯功能性 UI

## 配色方案
| 角色 | 色值 | 意象来源 |
|------|------|----------|
| 背景主色 | slate-950 | 监控室暗场 |
| 卡片/面板 | slate-900 | 控制台面板 |
| 安全/通过 | emerald-500 | 安检绿灯 |
| 警告/中风险 | amber-500 | 机场黄灯 |
| 危险/拦截 | red-500 | 安检红灯 |
| 主操作色 | cyan-500 | 扫描光束 |
| 文字主色 | slate-100 | 暗场高对比文字 |
| 文字辅助 | slate-400 | 次级信息 |

## 字体排版
- 中文：系统默认 (PingFang SC / Microsoft YaHei)
- 数据/代码：JetBrains Mono 或 ui-monospace
- 排版节奏：紧凑行距，数据区域用 mono 对齐

## 动效与交互
- 扫描动画：消息发送后出现 cyan 脉冲扫描条
- 风险弹出：拦截时红色边框闪烁 2 次
- 页面切换：fade-slide 200ms ease-out
- 悬停：卡片 subtle 亮度提升 + scale(1.01)

## 页面结构
- 左侧窄侧栏（图标+文字导航）
- 主内容区：对话 / 日志 / 仪表盘 / 策略 四个模块
- 对话界面右侧嵌入实时检测面板

## 设计禁忌
- 禁止渐变按钮、3D 卡片、毛玻璃等装饰效果
- 禁止使用卡通图标，只用线性/极简图标
- 禁止大面积白色背景
- 禁止彩色大字标题，标题用 slate-100
